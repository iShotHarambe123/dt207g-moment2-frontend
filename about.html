Om webbapplikationen

Syfte

Denna webbapplikation är utvecklad för att hantera arbetserfarenheter. Den tillåter användare visa, lägga till och ta bort arbetserfarenheter från en databas via ett REST API.

Teknisk information

Frontend: HTML, CSS, JavaScript

Backend: Node.js, Express

Databas: SQLite

API: REST API med JSON-format

Databasstruktur

Applikationen använder SQLite som databasserver med följande tabell:

Tabell: workexperience

  • id: Primärnyckel (auto-increment)
  • companyname: Företagsnamn
  • jobtitle: Jobbtitel
  • location: Arbetsplats
  • startdate: Startdatum (YYYY-MM-DD)
  • enddate: Slutdatum (valfri)
  • description: Arbetsbeskrivning
  • created_at: Tidsstämpel för skapande

API-endpoints

  • GET /api/workexperience - Hämta alla arbetserfarenheter
  • POST /api/workexperience - Skapa ny arbetslivserfarenhet
  • DELETE /api/workexperience/:id - Ta bort arbetslivserfarenhet

Slutsatser

Genom att utveckla denna applikation har jag lärt mig att:

  • Skapa REST API:er med Node.js och Express
  • Använda SQLite som databas
  • Implementera CRUD-operationer
  • Använda Fetch API för frontend-backend kommunikation
  • Hantera asynkron JavaScript
  • Validera data på både klient och serversidan
